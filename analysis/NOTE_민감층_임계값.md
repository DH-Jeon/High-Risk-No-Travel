# 민감층(아이·부모님) 점수 반영 방식 — 배율에서 임계값 하향으로 (2026-07-13)

## 문제
프로필 가중이 감점에 대한 배율(아이: 폭염·미먼 ×1.3)이라, **감점이 0인 온화한 날엔
프로필을 바꿔도 점수가 전혀 변하지 않았다** (0 × 1.3 = 0). 사용자에겐 기능 고장처럼 보임.

## 표준 위험지수 조사 결과
국내외 표준은 모두 "민감군은 **더 낮은 임계값**에서 경고 시작" 구조를 쓴다 (배율 방식 없음):

| 시스템 | 구조 |
|---|---|
| 기상청 폭염 영향예보 (2019~) | 보건 분야에서 일반인/취약계층(어린이·노약자) 위험수준을 별도 발표. 취약계층은 최고 체감온도 **31℃** 2일 지속부터 "관심"(피해 예상) — 일반 주의보(33℃)보다 2℃ 낮은 진입점 |
| 미국 NWS HeatRisk | 5단계 중 Level 1 = "폭염에 극히 민감한 사람 영향", Level 2 = "민감군 대부분 영향" — 민감군이 낮은 단계에서 먼저 걸리는 계단식 |
| 미국 EPA AQI | 101~150 = "Unhealthy for Sensitive Groups" 전용 구간 — 같은 농도에서 민감군만 경고 |

## 채택한 스펙 (weights.ts)
- **폭염**: 아이·부모님 동반은 감점 곡선을 **2℃ 왼쪽 이동** (주의보 33→31℃, 램프 시작 28→26℃).
  근거 = 영향예보 취약계층 관심단계(31℃). 온화한 날에도 프로필 간 차이가 생긴다.
- **미세먼지**: 아이 동반은 민감군 곡선 — 보통 3→5점, 나쁨 8→12점 (좋음은 0 유지).
  근거 = AQI USG 구조. ×1.3 배율은 제거.
- **부모님 동반 폭염 민감 신설**: 기존엔 ×1.0(무반응)이었으나, 영향예보 취약계층에
  노약자가 포함되므로 아이와 동일한 2℃ 하향 적용. 의료 ×1.5는 유지(접근성은 배율이 적절).
