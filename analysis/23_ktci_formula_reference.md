# 23. 관광기후지수(TCI/KTCI) 산출식 레퍼런스 — 쾌적층 계산 근거

출처:
- 기상청 맞춤형 관광기후지수 서비스 (bd.kma.go.kr, 전국 299 시군구 K-TCI 제공, 특보발령주의 등급 내장)
- 박창용 외(2014), "TCI를 이용한 치악산 국립공원 관광기후환경", 대한지리학회지 49(5) — **TCI 공식·변환표 원문**
- 김남조 외, "한국형 관광기후지수(KTCI) 개발" (KCI) — **한국형 실증 가중치**
- Mieczkowski(1985), TCI 원논문

## TCI 공식
```
TCI = 2(4·Cid + Cia + 2·R + 2·S + W) = 8·Cid + 2·Cia + 4·R + 4·S + W   (최대 100)
```
각 세부지수는 -3.0 ~ 5.0점.

## 세부지수 5개 + 변환식

| 지수 | 입력 | 변환 규칙 | 가중 |
|---|---|---|---|
| **Cid** 주간쾌적 | 최고기온+최저습도 | ASHRAE 유효온도 격자표 → 점수(별도) | 40% |
| **Cia** 일쾌적 | 평균기온+평균습도 | 같은 격자표 | 10% |
| **R** 강수 | 일강수량 | ≥5mm→0.0 / 0.5mm 감소마다 +0.5 / <0.5mm→5.0 | 20% |
| **S** 일조 | 일조시간 | ≤1h→0.0 / 1h당 +0.5 / >10h→5.0 | 20% |
| **W** 바람 | 풍속(km/h) | 아래 표(normal system) | 10% |

**바람(W) 변환표** (Mieczkowski normal system):
| 풍속 km/h | 점수 |
|---|---|
| <2.88 | 5.0 |
| 2.88–5.75 | 4.5 |
| 5.76–9.03 | 4.0 |
| 9.04–12.23 | 3.5 |
| 12.24–19.79 | 3.0 |
| 19.80–24.29 | 2.5 |
| 24.30–28.79 | 2.0 |
| 28.80–38.52 | 1.0 |
| >38.52 | 0 |

**Cid/Cia 격자표**: 최고기온×최저습도(Cid), 평균기온×평균습도(Cia)의 ASHRAE 유효온도 격자.
→ 원논문 그림1(격자표) 필요. **우리는 이미 `feelsLikeSummerC`(체감온도)를 계산하므로, 체감온도→Cid/Cia 점수 곡선으로 브리지 가능** (여름 고온다습에서 급감하는 특성 동일).

## 등급표 (Mieczkowski)
90–100 ideal / 80–89 excellent / 70–79 very good / 60–69 good / 50–59 acceptable
/ 40–49 marginal / 30–39 unfavorable / 20–29 very unfav / 10–19 extremely / <9 impossible

## 한국형(KTCI) 실증 가중치 (4계절 통합) — 원 40/10/20/20/10 대체
| 요소 | KTCI 가중 | (원 TCI) |
|---|---|---|
| 최고기온(Cid) | **25.9%** | 40% |
| 평균기온(Cia) | **20.9%** | 10% |
| 강수 | **32.2%** | 20% |
| 풍속 | **11.6%** | 10% |
| 구름/일사 | **9.5%** | 20% |
→ 한국은 **강수 비중 최고(32%)**. 우리는 KTCI 가중 채택 권장.

## 우리 서비스 적용안
1. **쾌적층 = KTCI** (한국형 가중, 위 변환식). 단기예보로 계산 가능(최고/평균기온·습도·풍속·강수·하늘상태SKY→일사 근사).
2. **미세먼지 추가** (KTCI엔 없음, 우리 확장): 6번째 축, 가중 ~15%로 넣고 5축을 85%로 재정규화. 근거=환경부 PM 등급.
   - 예: Cid 22 / Cia 18 / 강수 27 / 풍속 10 / 일사 8 / 미세먼지 15 = 100
3. **override**: 특보(호우/산불/산사태 경보) → '특보발령주의'로 강제 하향 (기상청 방식 동일).
4. **안전층(병원·고립)**: TCI/KTCI엔 없는 우리 차별화 — 취약성 감점(설계값, 지역안전지수 구조).

## 한계 (논문 명시 — 정직하게 반영)
- TCI는 **관광객 수를 예측하지 못함** (치악산: 여름 휴가철엔 TCI 낮아도 방문 급증, 단풍철 등 특정시기 불일치). → 우리의 "사고·방문수로 검증 실패" 발견과 일치. TCI는 **쾌적도 지표일 뿐**.
- 동계스포츠·연령/지역별 선호 미반영.
