# 하리노트 (HARI-NOTE: High Risk, No Travel)

> 강원 가족 여행자를 위한 관광지 안전 리스크 점수화 + 안전 대체 코스 추천 웹서비스
> 『2026 관광데이터 활용 공모전』 ① 웹·앱 개발 부문 출품작 — 팀 새벽코딩

## 한 줄 소개

기존 여행 앱이 "강원 어디가 예뻐요"를 말한다면, 하리노트는 **"오늘 이 강원 관광지는 가족 여행에 주의 요인이 높으니, 비슷한 만족도의 더 안전한 대체지로 바꾸세요"**를 말합니다.

## 팀

| 역할 | 담당 |
|------|------|
| 기획 · 백엔드 · 데이터 연동 · 지도 | 전동현 |
| 안전 점수 모델 · 프론트엔드 · 시각화 | 박현준 |

## 핵심 기능

1. **안전 점수 (0~100)** — TourAPI 강원 관광지별 폭염·강수·산불·미세먼지·의료·이동 위험을 공공기관 공식 임계값 기반 가중합으로 점수화
2. **위험 요인 분해** — "왜 주의해야 하는가"를 막대그래프로 시각화
3. **안전 대체 코스** — 거리·카테고리 유사도·점수·실내외 적합도 기반 대체 관광지 + 음식점 + 문화시설 반나절 코스
4. **가족·시니어 모드** — 아이/부모님 동반, 자차/뚜벅이별 위험 가중치 차등 적용
5. **출발 전 체크 리포트** — 산악기상·산불·통제·응급의료 한 장 요약

## 데이터 파이프라인

```
TourAPI(KorService2) 강원 관광지 → 외부 공공데이터 결합(기상청·산림청·AirKorea·복지부·행안부)
→ 위험 정량화 → SafetyScore 산출 → 위험 유형 분류 → 대체지 매칭 → 화면 출력
```

## 기술 스택

- **Frontend/Backend**: Next.js (App Router, TypeScript, Tailwind CSS)
- **데이터 저장**: TourAPI 수집 데이터를 앱에 JSON으로 내장 (MVP는 무장애 경량 구조 — 확장 시 Supabase 도입, `docs/decisions/004` 참고)
- **배포**: Vercel (GitHub 연동 자동 배포)
- **지도**: Leaflet + OpenStreetMap (API 키·사업자 등록 불필요)
- **데이터**: 한국관광공사 TourAPI 4.0 + 기상청 + AirKorea + 산림청 + 행정안전부 + 보건복지부

## 데이터 활용

### 내장 정적 데이터 (배치 수집 → JSON으로 저장소에 포함)

| 데이터 | 원천 | 용도 |
|---|---|---|
| 강원 관광지 2,091건 | 한국관광공사 TourAPI 4.0 (`KorService2`, areaCode=32) | 관광지 779·문화시설 109·음식점 1,203건의 이름·주소·좌표·시군코드·사진. 모든 화면의 기본 데이터 |
| 응급의료기관 22곳 | 국립중앙의료원 헬스맵 공공보건의료 통계 (공공데이터포털) | 관광지별 최근접 응급실 거리를 Haversine으로 실계산 — 네트워크·키 불필요 |
| 강원 시군 경계 | 통계청 2013 행정경계 GeoJSON (southkorea-maps) | 홈 안전지도의 시군 choropleth + 강원 외 지역 마스크 |

### 실시간 API (요청 시 서버 호출, 시군 대표점 18곳 + 1시간 캐시)

| API | 가져오는 값 |
|---|---|
| 기상청 단기예보 | 기온·강수확률·강수량·풍속 (위경도→기상청 격자 변환 후 조회) |
| AirKorea | 강원 실시간 PM2.5 (측정소 40곳→시군 매핑) |
| 산림청 산불위험예보 V2 | 시군별 산불위험지수 4단계 (활용신청 승인 시 자동 전환) |
| OpenStreetMap 타일 | Leaflet 지도 배경 (키 불필요) |

> ⚠️ 본 서비스의 안전 점수는 공공데이터 기반 **참고 정보**이며, 안전을 보장하지 않습니다.
